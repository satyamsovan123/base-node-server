(async, static) compare(data, hash) → {boolean}

Description:
  • This function compares given data i.e. password with the hash from the database. It is wrapped in a try-catch block to catch any errors. If an error occurs, false is returned and the error is logged. Contrary to the belief, this function does not decrypt the hash. It hashes the plain text data and compares it with the hash. This might take some time depending on the number of rounds used to generate the hash. So, it is wrapped in an async function.
Source:
  • utils/passwordUtils.js, line 37
Example
const result = await compare("password", "$2b$10$Z3..."); // true
Parameters:
Name Type Description
data string is the plain text data i.e. password to be compared.
hash string is the hash retrived from the database.
Returns:
true if the data matches the hash, false otherwise.
Type
boolean
